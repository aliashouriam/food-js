# Food JS

Food JS یک کتابخانه سبک، سریع و قابل توسعه برای مدیریت و نمایش داده‌های مرتبط با غذا است.  
این پروژه به شما امکان می‌دهد فهرست غذا‌ها را مدیریت کنید، آن‌ها را براساس دسته‌بندی فیلتر کنید، اطلاعات تغذیه‌ای بسازید و رابط‌های کاربری پویا ایجاد کنید.

---

## ویژگی‌ها

- معماری ساده و ماژولار
- بدون وابستگی خارجی (Vanilla JS)
- مناسب برای وب‌اپ‌های مدیریت رستوران، منو، یا فروشگاه غذایی
- پشتیبانی از دسته‌بندی غذاها (غذای ایرانی، فست‌فود، نوشیدنی و غیره)
- قابلیت جست‌وجو و فیلتر حرفه‌ای
- امکان افزودن داده‌های تغذیه‌ای مانند کالری، پروتئین، چربی و...
- مناسب برای ساخت منوهای پویا یا API سمت کلاینت
